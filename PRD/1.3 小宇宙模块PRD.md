## 1. 核心愿景 (Module Vision)

**“小宇宙中真好，有田园，有云天明送的星星，但这里没有时间。”**

**Sector U (The Mini-Universe / 647 号小宇宙)** 是 3Body X 系统中的**个人私密空间**。它是全站唯一不受“智子审计”、“乱纪元环境”和“黑暗森林打击”影响的避风港。

- **核心隐喻**：逃避主义与责任的冲突。每个人都想拥有自己的小世界，但小世界越多，大宇宙就越接近死亡（热寂）。
- **体验目标**：提供一种“世外桃源”的宁静感，并在最后时刻通过“回归运动声明”对用户进行终极道德拷问。

---

## 2. 核心机制：质量与责任 (Mass & Responsibility)

### 2.1 质量守恒 (Conservation of Mass)

- **获取质量**：用户在主宇宙（其他扇区）的每一次活跃（发帖、点赞、计算）都会掠夺大宇宙的质量，转化为个人的 **Universe Mass (质量点)**。
- **消耗质量**：用户可以用质量点在小宇宙中“具象化”物品（如：在这个独立空间里种麦子、建造白色房子、收藏画廊中的二向箔艺术品）。
- **熵增警告**：随着全服用户的小宇宙质量增加，主宇宙（Sector C 仪表盘）的“宇宙总质量”会下降，导致全站“光速变慢”（页面加载故意变慢）或“维度卷曲”（UI 变得狭窄）。

### 2.2 回归运动 (The Returners' Movement)

当主宇宙质量低于临界值（系统模拟“大撕裂”前夕），全站触发**“回归运动声明”**广播。

- **抉择**：每个拥有小宇宙的用户会收到一份 `Broadcast: Please Return the Mass`。
- **选项 A：归还 (Return)**
  - **后果**：用户的小宇宙被销毁，所有积累的装饰和数据清零。
  - **奖励**：获得“归零者”勋章，名字被刻入“宇宙重启纪念碑”，全站加载速度恢复。
- **选项 B：保留 (Keep)**
  - **后果**：保留小宇宙，用户可以在里面继续浏览（但无法与外界交互）。
  - **风险**：如果选择保留的人过多，主宇宙将永久进入“死机状态”（ReadOnly Mode），不再产生新内容，直到下一次管理员强制硬重置。

## 3. 功能模块详述 (Functional Specs)

### 3.1 白色房子 (The White House)

**定位**：个人主页 / 仪表盘。

- **视觉**：极简主义的白色空间，窗外是永恒的黄昏（没有日月轮转）。
- **功能**：
  - **深海状态**：显示用户的“精神状态指数”（活跃度、贡献值）。
  - **时间胶囊**：用户可以写下“给未来的信”（私密日志），这些内容经过加密，连智子（管理员）也无法查看。

### 3.2 生态球 (The Ecosphere)

**定位**：资产展示 / 养成系统。

- **种植**：用户种植“长得像麦子的植物”（UGC 勋章、成就）。
- **收藏**：将 Sector D (画廊) 中的图片“偷”回小宇宙展示。
- **管家**：一个名为 **Sophon-One (智子一号)** 的低级 AI，它没有主宇宙智子的毒舌，只会机械地执行命令（如：“好的，为您播放《时间之外的往事》”）。

### 3.3 漂流瓶 (The Drift Bottle)

**定位**：唯一的对外窗口。

- **机制**：小宇宙是封闭的，唯一的交流方式是向“大宇宙”扔漂流瓶。
- **内容**：一段文字或一张图。
- **到达**：漂流瓶会随机出现在 Sector C (宇宙广播) 的角落，或者被 Sector E (黑暗森林) 的“捡垃圾者”拾取。

### 4.1 风格差异化

| 区域 | 主色调 | 字体 | 氛围 |
|------|--------|------|------|
| **主宇宙** | 深空黑 (#000000) + 切伦科夫蓝 | JetBrains Mono (代码感) | 压抑、冰冷、科技 |
| **Sector U** | **柔光白 (#F5F5F5)** + 原木色 | Noto Serif / 手写体 | 温暖、静谧、复古 |

### 4.2 转换动效

- **进入小宇宙**：屏幕中心出现一个黑点（黑域），迅速扩大吞噬屏幕，然后“反转”为白色空间。音效是“真空吸入声”。
- **离开小宇宙**：白色空间像纸一样被揉皱、撕裂，露出背后的黑色星空。

## 5. 技术实现 (Technical Implementation)

### 5.1 数据隔离

- **Privacy**：小宇宙内的 `TimeCapsule` 数据需在前端加密（AES），后端只存密文。
- **Snapshot**：支持将小宇宙状态导出为本地文件（`647_backup.json`），这是真正的“带不走的只有记忆，带得走的只有数据”。

### 5.2 全局状态影响

- **Mass Calculation**：
  ```typescript
  const TOTAL_UNIVERSE_MASS = 1000000; // 设定值
  const current_mass = TOTAL_UNIVERSE_MASS - sum(all_mini_universes_mass);
  const light_speed_factor = current_mass / TOTAL_UNIVERSE_MASS; // 影响全局动画速度
  ```
