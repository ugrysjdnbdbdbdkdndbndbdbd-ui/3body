## 1. 核心愿景 (Module Vision)

**“不要回答！不要回答！不要回答！”**

**Sector R (The Red Coast Console / 红岸控制台)** 是 3Body X 系统的**信号收发与博弈中心**。它是一个复古的 SETI（搜寻地外文明）终端，用户在这里监听来自宇宙深处的噪音，并决定是否向黑暗森林发出自己的声音。

- **核心隐喻**：沟通即暴露。每一次交流都是在拿生存做赌注。
- **体验目标**：通过极简的命令行交互和压抑的复古音效，营造出“人类第一次接触外星文明”时的战栗感。

---

## 2. 交互与视觉设计 (UX & Visual)

### 2.1 风格定义 (The Retro Style)

- **界面**：完全复刻 1970s 的 **CRT 显示器风格**。
  - **色彩**：单色磷光绿 (`#00FF41`) 或 琥珀黄 (`#FFB000`)，黑色背景。
  - **特效**：屏幕会有随机的噪点、扫描线抖动、鱼眼畸变。
  - **字体**：点阵字体 (Pixel Art Font)。
- **音效**：
  - 背景音：低频的电流嗡嗡声、巨大的抛物面天线转动的机械声。
  - 操作音：老式机械键盘的敲击声、继电器吸合的咔塔声。

### 2.2 操作逻辑 (The Console Interaction)

- **无鼠标模式**：鼓励（或强制）使用键盘指令。
  - `LIST`：查看频道。
  - `TUNE <freq>`：调频。
  - `DECODE <msg_id>`：尝试解码。
  - `BROADCAST <content>`：发射信号。

## 3. 核心机制：信号博弈 (Signal Game)

### 3.1 监听与解码 (Listening)

- **宇宙背景辐射**：屏幕上持续滚动着乱码（瀑布流）。
- **发现信号**：偶尔会出现一段“高亮”的波形。用户需要手动调整“频率旋钮”（UI 组件）来对齐波形。
- **解码挑战**：
  - 信号内容是加密的（可能是 AI 生成的谜题，也可能是其他用户发送的密文）。
  - 用户需要消耗“算力”或通过简单的逻辑解谜来还原信息。
  - **内容示例**：“我是墓地老人，不要去二向箔画廊，那里有埋伏。”

### 3.2 发射与代价 (Broadcasting)

用户可以编写一条信息，选择发射功率。

- **微波发射 (Low Power)**：
  - **范围**：仅好友或随机少数人可见。
  - **代价**：消耗少量积分。
  - **风险**：低。

- **引力波广播 (High Power / The Ye Wenjie Button)**：
  - **范围**：全站广播（Sector C 广播站置顶，所有用户收到通知）。甚至可对接真实世界的 Twitter/Discord Bot。
  - **操作**：需要输入复杂的“发射授权码”，并经过 3 次“后果确认”。
  - **代价**：
    - 账号进入 **"Exposed" (坐标暴露)** 状态 24 小时。
    - 暴露状态下，该用户在 Sector E (黑暗森林) 中无法隐身，发帖会被高亮标记，极易遭受“二向箔打击”。
  - **隐喻**：为了传播真理，必须牺牲安全。

### 3.3 智子干扰 (Sophon Interference)

- **锁死**：智子会随机“锁死”控制台，屏幕显示“物理学不存在”。
- **篡改**：用户接收到的信号可能被智子实时篡改。
  - *原信*：“主不在乎。”
  - *篡改后*：“主很在乎你，请发送坐标。”
- **识别**：用户需要通过逻辑判断（RAG 知识库对比）来识破智子的谎言。

## 4. 功能模块详述 (Functional Specs)

### 4.1 频率瀑布图 (Waterfall Plot)

- **可视化**：实时 WebGL 渲染的频谱图。
- **交互**：鼠标拖拽或键盘左右键移动“监听窗口”。
- **彩蛋**：在特定频率（如 `1974.11.16` Arecibo Message）可能会收到特殊的历史彩蛋。

### 4.2 红色按钮 (The Red Button)

- **视觉**：一个带有物理保护盖的红色按钮 UI。
- **交互**：
  1. 点击保护盖 -> 盖子弹开。
  2. 长按按钮 -> 蓄力条上涨（伴随警报声）。
  3. 松开 -> 发射。
- **后果**：屏幕瞬间变红，显示“BROADCASTING...”，随后全站所有用户的屏幕出现短暂的信号干扰抖动。

## 5. 数据结构 (Data Schema)

```typescript
interface Signal {
  id: string
  source_type: 'user' | 'ai' | 'system'
  frequency: number // MHz
  content_encrypted: string
  content_decrypted: string
  power_level: 'low' | 'high'
  integrity: number // 信号完整度，受智子干扰影响
  timestamp: number
}

interface UserConsoleState {
  current_frequency: number
  is_exposed: boolean // 是否暴露坐标
  exposed_until: number // 暴露结束时间
  decryption_skill: number
}
```
