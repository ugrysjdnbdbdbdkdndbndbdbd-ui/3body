## 1. 核心愿景 (Module Vision)

**“在宇宙的黑暗森林中，每个人都是一道折射出文明光辉或阴暗的棱镜。”**

**Sector P (The Prisms / 人类棱镜)** 不是简单的百科全书，而是一个**动态的人性观测站**。它展示三体宇宙中关键人物（面壁者、破壁人、执剑人、普通人）的心理画像、关键决策及其引发的蝴蝶效应。

- **核心隐喻**：人物即**棱镜**。宇宙公理（光）穿过他们，折射出不同的历史走向（光谱）。
- **体验目标**：用户不仅是在“阅读”人物，更是在“解析”他们的精神世界和因果权重。

## 2. 交互与视觉设计 (UX & Visual)

### 2.1 入口：面壁者长廊 (The Wallfacer's Corridor)

- **视觉风格**：无尽的长廊，两侧悬浮着被“琥珀”包裹的人物全息投影。
- **交互**：
  - 滚动浏览时，人物会有微弱的“呼吸感”动效。
  - 鼠标悬停，琥珀表面会出现裂纹（象征破壁）或高亮（象征执剑），并浮现该人物的**核心威慑度**数值。

### 2.2 详情页：思维宫殿 (The Mind Palace)

- **布局**：非线性布局，以人物为中心，向四周发散出“决策树”。
- **动态立绘**：
  - 支持 Live2D 或 3D 模型，人物表情随“当前浏览的纪元”而变化（如罗辑在危机纪元的玩世不恭 vs 威慑纪元的冷酷如冰）。
  - **视网膜干扰**：智子会在立绘旁实时弹幕吐槽（如：“这个虫子很危险”）。

## 3. 功能模块详述 (Functional Specs)

### 3.1 人格光谱分析 (Personality Spectrum)

摒弃传统的战力/智力雷达图，采用**三体特色指标**：

- **异化指数 (Alienation Index)**：由 0%（常人）到 100%（非人/神性）。
  - *例：维德 95%，程心 5%，罗辑（后期）90%*
- **威慑值 (Deterrence Value)**：对三体世界的心理压迫力。
- **爱意值 (Empathy/Love)**：对人类整体或个体的执念。
- **公理契合度**：该人物行为逻辑与“宇宙公理”的吻合程度（关联 3.0 零点计划）。

### 3.2 因果纠缠图谱 (The Entanglement Graph)

展示人物与其他角色的关系，以及对历史节点的干涉。

- **强连接**：实线，如“罗辑 - 庄颜”。
- **弱连接/猎人与猎物**：虚线/红外线，如“罗辑 - 三体人”。
- **观测者连线**：用户（作为观测者）与该人物的互动记录。

### 3.3 关键决策复盘 (Decision Replay)

与 **Sector B (编年史)** 深度联动。

- **节点重现**：列出该人物一生的关键转折点（如：叶文洁按下发射按钮、章北海谋杀老航天）。
- **如果（What If）**：关联 **3.0 零点计划** 的因果模拟。用户可点击“如果他不这么做”，系统基于逻辑引擎推演平行时间线摘要。
  - *例：如果章北海没有按兵不动？ -> 智子推演：人类舰队在 100 年前即被毁灭。*

### 3.4 跨时空对话 (Echo Interaction)

- **AI Persona**：基于 RAG 和人物性格设定的 AI 对话。
- **记忆保留**：AI 记得用户在 Sector A（智子）中的提问记录，如果用户曾对智子表现出敌意，罗辑可能会表示赞赏，程心可能会表示担忧。

## 4. 数据结构 (Data Schema)

```typescript
interface CharacterProfile {
  id: string
  name_cn: string
  name_en: string
  alias: string[] // ["面壁者", "执剑人", "墓地老人"]
  
  // 视觉资产
  avatar_url: string
  full_body_model_url?: string // Live2D / 3D
  theme_color: string // 专属色，如 罗辑-冰蓝，章北海-军绿
  
  // 三体维度指标
  metrics: {
    alienation: number // 异化
    deterrence: number // 威慑
    empathy: number    // 爱
    rationality: number // 绝对理性
  }
  
  // 关键纪元状态 (随时间轴变化)
  eras: {
    [era_name: string]: {
      status: string // "冬眠", "苏醒", "执剑", "死亡"
      quote: string  // 该时期的名言
      description: string
    }
  }
  
  // 因果节点 (关联 Chronicle)
  key_events: {
    event_id: string // 关联 Sector B Event ID
    role: "trigger" | "victim" | "observer" // 触发者/受害者/观察者
    impact_description: string
  }[]
}
```
