## 1. 核心愿景 (Module Vision)

**“给岁月以文明，给时光以记忆。”**

本次重构旨在将 **Sector B (编年史)** 从一个简单的“时间轴列表”升级为**三体宇宙的终极知识库**。它将承载从微观粒子到宏观宇宙的所有历史记录、官方档案与科学定义。

- **定位升级**：Timeline (线性) -> Archive (深度) + Encyclopedia (广度)。
- **核心价值**：为用户提供从“发生了什么”（Timeline），到“背后的真相”（Archive），再到“相关概念”（Encyclopedia）的完整认知链路。

---

## 2. 架构拆分 (Architecture Split)

Sector B 将包含三个并行的子视图：

### 2.1 视图 A：时间之流 (The Timeline View)
*保留原有功能并优化*

- **核心功能**：以时间为轴，线性展示宇宙大事记（魔法纪元 -> 威慑纪元 -> 掩体纪元 -> 银河纪元）。
- **交互优化**：
  - **无极缩放**：支持从“亿年”尺度（宇宙热寂）缩放到“秒”尺度（古筝行动倒计时）。
  - **因果节点**：点击节点可跳转至关联的【宇宙档案馆】深度文章或【三体百科】词条。

### 2.2 视图 B：宇宙档案馆 (The Cosmic Archive)
*新增 PGC 深度内容*

- **定位**：官方/COO 发布的**绝密档案**。
- **内容形式**：
  - **红岸解密**：叶文洁的原始审讯记录、红岸工程技术文档。
  - **面壁计划**：四位面壁者的详细战略白皮书。
  - **阶梯计划**：云天明大脑飞行轨迹的监测报告。
- **权限管理**：
  - **COO/Admin**：拥有 `Creator` 权限，可创建、编辑、封存档案。
  - **User**：拥有 `Viewer` 权限，需消耗积分或达到特定等级（如“ETO 核心层”）才能解密阅读。
- **视觉风格**：
  - 拟物化文件袋、绝密印章 (`TOP SECRET`)、打字机字体。
  - 附件支持：音频（录音带）、图片（胶卷底片风）、蓝图（工程图纸）。

### 2.3 视图 C：三体百科 (The Encyclopedia)
*新增结构化概念库*

- **定位**：三体宇宙的**维基百科**。
- **分类体系**：
  - **文明 (Civilizations)**：三体文明、地球文明、歌者文明、归零者、魔戒文明。
  - **技术 (Technology)**：曲率驱动、水滴（强互作用力）、智子、二向箔、光粒。
  - **现象 (Phenomena)**：宇宙闪烁、乱纪元、降维打击、黑域（慢雾）。
  - **设施 (Facilities)**：红岸基地、亚洲舰队、掩体世界。
- **展示形式**：
  - **卡片式浏览**：瀑布流展示概念卡片。
  - **沉浸式详情**：
    - *歌者文明*：页面背景会有低吟的歌声，文字呈诗歌排版。
    - *四维碎片*：嵌入 WebGL 控件，允许用户旋转查看“克莱因瓶”或“莫比乌斯环”结构。
    - *太阳系二维化*：页面滚动时，文字和图片逐渐“压扁”，模拟降维过程。

## 3. 功能模块详述 (Functional Specs)

### 3.1 档案编辑器 (Archive CMS for COO)

为 COO 提供一个**可视化编辑器**：
- **元数据**：标题、密级（绝密/机密/公开）、归档日期、关联时间节点。
- **富文本**：支持 Markdown，支持插入“智子备注”（只有特定用户可见的隐藏文字）。
- **附件上传**：上传图片/音频，系统自动处理为“旧档案风格”（添加噪点、折痕）。

### 3.2 百科关联图谱 (Wiki Graph)

在百科详情页底部，展示该词条的**关联网络**：
- *例如查看【水滴】* -> 关联人物【丁仪】 -> 关联事件【末日战役】 -> 关联技术【强互作用力】。
- 使用 **D3.js / ECharts** 渲染力导向图。

## 4. 视觉与交互规范 (Visuals)

### 4.1 档案馆风格 (Archive Style)

- **背景**：深灰色的金属档案柜质感，或磨损的牛皮纸纹理（根据密级不同）。
- **字体**：`Courier New` 或 `Special Elite` (打字机风格)。
- **动效**：打开档案时，有“拆开封条”的动画和“纸张摩擦”的音效。

### 4.2 百科风格 (Wiki Style)

- **背景**：全息投影风格，半透明玻璃态卡片。
- **高亮**：关键术语（如“黑暗森林”）高亮显示，鼠标悬停显示微型浮层解释。

## 5. 数据结构 (Data Schema)

```typescript
// 宇宙档案馆
interface ArchiveDocument {
  id: string
  title: string
  classification: 'TOP_SECRET' | 'CONFIDENTIAL' | 'PUBLIC'
  content_markdown: string
  attachments: { type: 'image' | 'audio', url: string }[]
  linked_event_id?: string // 关联 Timeline
  created_by: string // COO User ID
  created_at: number
}

// 三体百科
interface WikiEntry {
  id: string
  term: string // 词条名
  category: 'Civilization' | 'Tech' | 'Phenomena' | 'Facility'
  summary: string
  content_markdown: string
  visual_effect?: 'flatten' | 'glitch' | '4d_rotate' // 特殊视觉效果
  related_entries: string[] // ID List
}
```

## 6. 结语

**“弱小和无知不是生存的障碍，傲慢才是。”**

本次重构将消除用户的“无知”。

通过 Timeline 串联历史，通过 Archive 揭示真相，通过 Encyclopedia 定义概念。Sector B 将成为三体宇宙中信息密度最高、最权威的**真理殿堂**。
