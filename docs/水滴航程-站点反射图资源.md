# 水滴航程 · 九站反射图资源

为「水滴航程」九个站点挑选的**推荐反射环境图**，用于 3D 镜面反射（HDR/Equirectangular）。当前实现使用 TresJS 内置 preset 作为占位；后续可改为自定义 HDR 以更贴合宇宙场景。

---

## 资源来源概览

| 来源 | 类型 | 许可 | 说明 |
|------|------|------|------|
| **Space Spheremaps** | 太空 HDR/球面图 | 免费商用 | 深空、星云、小行星带、行星等分类，equirectangular |
| **Poly Haven** | HDRI (HDR/EXR) | CC0 | 夜空、室内、城市等，多分辨率，直接下载 URL |
| **Tresjs Assets** | 内置 preset | 项目依赖 | 当前使用的 night/city/studio/sunset/dawn/snow/modern |

## Poly Haven 直接下载 URL 格式

- **EXR 2K**：`https://dl.polyhaven.org/file/ph-assets/HDRIs/exr/2k/{asset_id}_2k.exr`
- **HDR 2K**：`https://dl.polyhaven.org/file/ph-assets/HDRIs/hdr/2k/{asset_id}_2k.hdr`
- **1K**：将 `2k` 改为 `1k`，文件名 `_2k` 改为 `_1k`。

常用 asset_id：`qwantani_night_puresky`、`venice_sunset_1`、`kiara_1_dawn`、`snowy_forest_path_01`、`modern_buildings_night`、`small_hangar_01`、`poly_haven_studio_1`、`canary_wharf_1`（city）。

## 集成说明（后续可做）

1. **当前**：`SectorDroplet.vue` 使用 `<Environment :preset="envPreset" />`，preset 来自 `WAYPOINTS[].preset`（Tresjs 内置）。
2. **自定义 HDR**：若需用上表推荐图，可：
   - 将对应 HDR/EXR 下载到 `frontend/public/hdri/droplet/`（或 CDN），
   - 使用 `<Environment :files="currentWaypointHdrUrl" />`（或 `path` + `files` 数组，依 TresJS Environment 文档），
   - 在 `Waypoint` 上增加字段如 `customEnvUrl?: string`，并优先于 `preset` 使用。

---

## Space Spheremaps 使用说明

- 站内分类：HDRi Space、Planetary、Asteroid Field、Blue Nebulae、Multi-nebulae、Stellar Phenomena、Plain Starfield 等。
- 下载后为 equirectangular 单图，需放入 `public/hdri/` 并以上述方式传入 Environment 的 `files`（单文件 HDR 即可）。
