# 三体外传创作规范 (Apocrypha Style & Canon Guide)

> 所有写入 `data/apocrypha.json` 或经本项目中「三体外传」栏目展示的同人/COO 内容，**必须**严格遵循本规范：三体行文风格 + 三体宇宙观。违反者不得入库。

---

## 一、行文风格 (Prose Style)

### 1.1 基调

- **冷静、克制**：叙述者不替角色抒情，不替读者下结论。情感通过动作、环境、细节呈现，不直接喊出「爱」「绝望」「伟大」。
- **白描多于议论**：少用「他感到……」「她意识到……」式的心理总结；多用可见可闻的描写（表情、动作、环境、技术细节）。
- **短句与留白**：可大量使用短句、断句、换行留白。避免长句堆砌和网络腔（「或许」「也许他们不会听」适度即可，不宜泛滥）。
- **科技与人文并重**：技术设定（智子、曲率、维度、光速）需准确；同时不把人物写成工具人，保持个体在宏大叙事下的具体选择与代价。

### 1.2 禁止

- **金句病**：避免刻意收束成「金句」式总结（如「结束才是开始」「她从来没有懂过」可偶用，但整篇不宜靠金句撑腰）。
- **上帝点评**：叙述者不评价角色「对/错」「伟大/渺小」，不替原著「补完主题」。
- **煽情与网络腔**：禁止「泪目」「破防」「意难平」式表达；禁止「或许这就是……吧」等轻浮收束。
- **架空口吻**：角色对话与内心独白需符合原著人设与时代，不出现现代网络用语或违和的热血/鸡汤腔。

### 1.3 推荐

- **具体优于抽象**：写「他把手从按钮上移开」优于「他放下了责任」。
- **因果清晰**：行为有动机、结果有代价，不为了「虐」或「燃」而扭曲人物逻辑。
- **与正典衔接**：时间锚、事件、技术节点与原著/零点计划设定一致，不发明新纪元、新法则。

## 二、宇宙观 (Canon Cosmology)

### 2.1 必须遵守的设定

- **黑暗森林法则**：生存是第一需要；文明不断扩张而物质总量不变；猜疑链 + 技术爆炸导致暴露即高危。
- **光速与维度**：光速不可超（曲率驱动是改变空间而非超光速）；维度只降不升；黑域 = 低光速 = 安全声明。
- **智子 / 水滴 / 二向箔**：智子锁死基础物理、全知但不解人心；水滴强互作用力材料、锐角转向；二向箔三维坍二维、不可逆。
- **面壁 / 破壁 / 执剑**：面壁者战略不可解读；破壁人揭穿战略；执剑人威慑度决定三体是否动手。
- **纪元与时间线**：黄金时代 → 危机纪元(2015起) → 威慑纪元 → 广播纪元 → 掩体纪元 → 银河纪元；乱纪元/恒纪元仅指三体母星，不随意套用地球。

### 2.2 禁止

- **发明新宇宙法则**：不新增「第X公理」、不发明与四大公理矛盾的设定。
- **技术越级**：某纪元尚未出现的技术不得提前出现（如危机纪元不得出现曲率、黑域成品）。
- **人物 OOC**：角色动机、选择与原著或已有人物志设定冲突即视为 OOC，需修正或弃用该情节。
- **与正典因果冲突**：外传可做「细节展开」或「续写」，但不得改写既定结果（如不得写「程心按下按钮」等颠覆正典结局）。

### 2.3 可接受范围

- **细节展开**：正典中一笔带过的场景（如「红岸那一夜」「执剑人交接的十分钟」）做合理扩写。
- **续写**：正典之后或之外的留白（如维德在牢中、关一帆笔记）可续写，但需与原著结局、归零者与质量归还等设定一致。
- **视角转换**：同一事件用不同角色视角（如 1379 号、歌者）重述，不改变事件结果。

## 三、与零点计划的关系

- **零点计划原创**（如守摆人、乱纪元 471 年）已纳入项目设定；外传中引用时保持与 `docs/Project-Zero-Axioms.md` 及现有档案馆/人物志一致。
- **因果律**：外传不要求像「文明碎片」一样走智子因果审计流程，但**逻辑自洽、不违背公理**为最低要求；若项目后续为外传增加审计，则本规范为审计依据之一。

---

## 四、自检清单（每条外传提交前）

- [ ] 无金句病、无上帝点评、无煽情/网络腔
- [ ] 时间锚与纪元、技术节点与正典一致
- [ ] 未发明新宇宙法则或技术越级
- [ ] 人物动机与选择符合原著/人物志
- [ ] 未改写正典既定结果（如威慑失效、二向箔、质量归还）
- [ ] 行文以白描与细节为主，冷静克制
