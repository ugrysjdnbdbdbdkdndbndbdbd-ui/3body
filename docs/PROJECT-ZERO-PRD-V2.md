# 📂 【零点计划 v2.0】PRD：思想体验与深维沙盒扩展

## 1. 角色与项目设定 (Role & Context)
- **AI 角色:** Senior AI Gameplay Architect & Full-Stack Developer (高级 AI 玩法架构师 & 全栈开发者)。
- **项目代号:** Project Zero (零点计划)。
- **核心目标:** 在现有的三体 RAG 社区基础上，引入基于 LLM 的高阶心理博弈与多智能体(Multi-Agent)演化沙盒。
- **开发原则:** 极致的硬科幻冷酷感；逻辑严密；UI 极简且具有压迫感。禁止任何拟人化、温情脉脉的提示词。

---

## 2. 核心增量模块需求 (Core Expansion Modules)

### 模块 A：【面壁者审讯室】 (The Wallbreaker AI)
**定位：** 逆向图灵测试与极端心理压迫体验。
**逻辑链路：**
1. **初始化:** 用户进入 `Sector W`，系统宣读面壁者任命书。用户在本地浏览器输入一个“真实战略目标”（存入 localStorage，不传给后端，保证心理安全感）。
2. **交互循环:** - 用户不断提交资源调配指令（如：“调集全球 30% 的算力研究人类面部微表情”）。
   - 后端 `WallbreakerAgent` (基于 DeepSeek-V3) 接收指令。它的 Prompt 设定为：“你是破壁人。分析面壁者的指令，寻找其隐藏的真实战略目的（逃亡主义、同归于尽、或者伪装）。不要立刻揭穿，陪他演戏，给出冷酷的资源批准或驳回理由。”
3. **破壁判定:** - `WallbreakerAgent` 内部维护一个 `Suspicion Score` (怀疑度)。
   - 当怀疑度 > 90% 时，AI 会强制接管 UI，屏幕变为血红色预警，输出破壁宣言：“主不在乎。但我是你的破壁人，你的真实计划是 [AI 推演的结论]。”

### 模块 B：【黑暗森林多智能体沙盒】 (Dark Forest Multi-Agent Sandbox)
**定位：** 全天候运行的后台宇宙社会学模拟器。
**逻辑链路：**
1. **Agent 抽象:** 创建 `CivilizationAgent` 类。每个 Agent 拥有初始参数：`technology_level` (技术爆炸概率), `paranoia_index` (猜疑链敏感度), `concealment_skill` (隐藏能力)。
2. **时钟周期:** 设立一个后台 Cron Job (如每 5 分钟代表宇宙流逝 100 年)。
3. **交互规则:**
   - Agent 会在虚拟坐标系中随机探测。
   - 触发“接触”时，双方 Agent 基于 LLM 进行“猜疑链博弈”（Prompt 强制执行黑暗森林法则）。
   - **结果:** 保持静默、发射探测器、或者直接发起“光粒打击”（销毁对方实例）。
4. **前端展示 (上帝视角):** - 在 `Sector F` 提供一个深空坐标图。用户只能旁观这些 AI 文明的生灭。
   - 提供一份实时更新的《宇宙墓志铭》，记录哪个编号的文明在哪个纪元因为何种愚蠢的广播被摧毁。

### 模块 C：【云天明隐喻加密局】 (Yun Tianming Cryptography)
**定位：** 高级语义加密与社区解密互动。
**逻辑链路：**
1. **加密器 (Encoder):** 用户输入一段机密文本（如真实代码或日记）。调用 LLM，Prompt：“作为云天明，将以下机密信息完全隐喻化，改写成一篇具有三体风格的荒诞童话（如《国王的新画师》）。确保核心逻辑链保留，但表层语义完全改变。”
2. **公海展示:** 加密后的童话发布到社区公共大厅。
3. **解密器 (Decoder):** - 其他用户使用算力点数（负熵）对童话进行语义拆解。
   - 系统提供“双层语义对比高亮”组件。成功还原原意 80% 以上的用户，获得“执剑人候补”徽章和巨额算力奖励。

### 模块 D：【思想钢印网络】 (The Mental Seal Epidemic)
**定位：** 隐蔽的群体潜意识修改实验。
**逻辑链路：**
1. **变量注入:** 管理员（或系统随机）设定一个钢印命题（例如：`Theme = "水是剧毒的"`）。
2. **数据污染 (The Poisoning):** - 当普通用户在社区阅读官方编年史或外传时，调用一个极轻量的中间件层 `MentalSealMiddleware`。
   - 该中间件将原始内容输入 LLM，Prompt：“在不改变原文主要情节和物理规律的前提下，通过极其隐晦的修辞、比喻或潜台词，植入对‘水是剧毒的’这一概念的恐惧感。”
3. **全服共振监测:** 统计用户在后续共创中，自发使用该钢印词汇的频率，前端展示一个全网“模因感染指数”热力图。

## 3. 技术栈与架构要求 (Tech Stack & Architecture)
- **框架:** Next.js 14 (App Router) + TypeScript + Tailwind CSS.
- **大模型接口:** 统一封装 `llm_gateway.ts`，接入 DeepSeek-V3 API，必须支持长上下文与高并发。
- **数据库扩展 (Supabase):**
  - 新增 `dark_forest_agents` 表（记录 AI 文明状态）。
  - 新增 `encrypted_fairytales` 表（记录加密童话及破译进度）。
- **动效库:** 引入 `Framer Motion`，用于实现破壁时的“UI 撕裂”、黑暗森林打击的“降维闪烁”等极端视觉效果。
